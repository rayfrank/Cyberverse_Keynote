# CYBERVERSE Keynote (Three.js)

This is a lightweight, browser-based keynote deck (5–6 slides) with a Three.js “stage” behind it.

## Quick start

1) Install dependencies:
- `npm i`

If PowerShell blocks `npm` (execution policy), use:
- `npm.cmd i`

2) Run:
- `npm run dev`

Or:
- `npm.cmd run dev`

## Customize

- Edit slide text/media/links in `src/deck.js`
- Drop assets into `public/assets/`
  - `public/assets/trailer.mp4`
  - `public/assets/shot1.jpg`

## Controls

- Next: `→`, `Space`, `PageDown`, or click
- Prev: `←`, `PageUp`
- Jump: `Home` / `End`
